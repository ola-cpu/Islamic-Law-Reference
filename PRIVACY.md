# Politique de confidentialité — Islamic Law Reference

*Dernière mise à jour : juin 2025*

## Résumé

Islamic Law Reference est conçue pour fonctionner **hors ligne**. Vos données personnelles restent en grande partie **sur votre appareil**.

## Données collectées

L'application peut stocker localement :

- Préférences (langue, thème, école juridique préférée, mode débutant/étudiant)
- Favoris, notes personnelles, progression (lecture, quiz, badges, parcours)
- Historique de recherche récent (sur l'appareil)
- Rappels de notification (si activés)

**Aucune donnée n'est envoyée automatiquement à nos serveurs** — l'application n'inclut pas de compte utilisateur centralisé ni d'analytics tiers par défaut.

## Sauvegardes et partage

- **Export JSON / PDF** : déclenché par vous via le menu partage du système.
- **Sauvegarde chiffrée** : chiffrement AES côté appareil avec votre phrase secrète ; le fichier peut être stocké où vous choisissez (Drive, iCloud, etc.).
- **Import** : vous choisissez un fichier ; l'app lit son contenu localement.

Nous n'avons pas accès à vos sauvegardes.

## Notifications

Si vous activez le rappel quotidien, l'app planifie une notification locale. Aucune donnée n'est transmise à un serveur pour cela.

## Widgets (Android / iOS)

Le widget « Sujet du jour » lit des données locales préparées par l'application.

## Liens externes

L'écran Méthodologie peut ouvrir des sites tiers (ex. SeekersGuidance, IslamQA) dans votre navigateur. Consultez leurs politiques respectives.

## Données des enfants

L'application est éducative et ne cible pas spécifiquement les enfants de moins de 13 ans. Aucune collecte intentionnelle de données d'enfants.

## Vos droits

Vous pouvez à tout moment :

- Supprimer vos données en désinstallant l'application
- Réinitialiser favoris/notes via les paramètres ou une réinstallation
- Ne pas activer les notifications ni les sauvegardes cloud

## Contact

Pour toute question : ouvrez une issue sur le dépôt GitHub du projet ou contactez le mainteneur indiqué sur la fiche Store.

## Modifications

Cette politique peut être mise à jour. La date en tête de document indique la dernière révision.
